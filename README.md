# Excel Data Formatter

Программа для форматирования данных Excel с графическим интерфейсом для Windows.

## Описание

Excel Data Formatter - это удобная программа с графическим интерфейсом, которая позволяет быстро и легко форматировать Excel файлы различными способами. Программа разработана специально для работы в среде Windows.

## Возможности

### Основное форматирование:
- **Форматирование заголовков** - делает первую строку жирной, синей с белым текстом
- **Автоподбор ширины колонок** - автоматически подстраивает ширину колонок под содержимое
- **Добавление границ** - добавляет границы ко всем ячейкам с данными
- **Форматирование чисел** - применяет числовой формат с разделителями тысяч
- **Центрирование текста** - выравнивает текст по центру во всех ячейках

### Дополнительные функции:
- **Удаление пустых строк** - удаляет все полностью пустые строки
- **Удаление дубликатов** - находит и удаляет дублирующиеся строки
- **Заморозка первой строки** - закрепляет заголовки при прокрутке
- **Сортировка по первой колонке** - автоматическая сортировка данных
- **Полное форматирование** - применяет все операции одновременно

## Системные требования

- Windows 7/8/10/11
- Python 3.7 или выше
- Excel файлы в формате .xlsx или .xls

## Установка

### 1. Установите Python

Скачайте и установите Python с официального сайта: https://www.python.org/downloads/

**Важно:** При установке отметьте галочку "Add Python to PATH"

### 2. Клонируйте репозиторий

```bash
git clone https://github.com/yourusername/G.git
cd G
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

Альтернативно, установите пакеты вручную:

```bash
pip install pandas openpyxl
```

## Использование

### Запуск программы

1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку с программой:
   ```bash
   cd путь\к\папке\G
   ```
3. Запустите программу:
   ```bash
   python excel_formatter.py
   ```

### Работа с программой

1. **Выбор файла:**
   - Нажмите кнопку "Обзор..."
   - Выберите Excel файл (.xlsx или .xls)

2. **Форматирование:**
   - Выберите нужную операцию форматирования
   - Нажмите соответствующую кнопку
   - Программа создаст новый файл с суффиксом "_formatted"

3. **Результат:**
   - Исходный файл остается без изменений
   - Новый файл сохраняется в той же папке с названием `имя_файла_formatted.xlsx`

### Пример

Если ваш файл называется `data.xlsx`, после форматирования будет создан файл `data_formatted.xlsx`

## Создание исполняемого файла (.exe)

Для создания .exe файла, который можно запускать без установки Python:

### 1. Установите PyInstaller

```bash
pip install pyinstaller
```

### 2. Создайте .exe файл

```bash
pyinstaller --onefile --windowed --name="ExcelFormatter" excel_formatter.py
```

### 3. Найдите готовый .exe

После завершения процесса найдите файл `ExcelFormatter.exe` в папке `dist/`

Теперь вы можете запускать программу двойным кликом по .exe файлу без необходимости установки Python!

## Советы по использованию

1. **Резервные копии:** Программа автоматически создает новый файл, но рекомендуется делать резервные копии важных данных

2. **Большие файлы:** Для файлов с большим количеством данных (>10000 строк) некоторые операции могут занять время

3. **Полное форматирование:** Используйте кнопку "Полное форматирование" для применения всех улучшений сразу

4. **Предварительный просмотр:** После форматирования откройте файл в Excel для проверки результата

## Устранение неполадок

### Ошибка "Python не найден"
- Убедитесь, что Python установлен
- Добавьте Python в PATH
- Перезапустите командную строку

### Ошибка "Модуль не найден"
- Установите все зависимости: `pip install -r requirements.txt`

### Программа не запускается
- Проверьте версию Python: `python --version`
- Обновите pip: `python -m pip install --upgrade pip`

### Файл не открывается
- Убедитесь, что файл имеет формат .xlsx или .xls
- Проверьте, что файл не поврежден
- Закройте файл в Excel перед форматированием

## Примеры использования
